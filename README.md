# cli-factory
Cli Factory - Built with ❤️

## Description
Cli Factory is a command line interface (CLI) tool designed to simplify the process of creating and managing command line applications. It provides a framework for building robust and user-friendly CLIs.

## Features
- Easy command creation
- Built-in help and documentation
- Support for plugins and extensions

## Installation
You can install cli-factory using npm:
```
npm install cli-factory
```

## Usage
To create a new CLI application, run:
```
cli-factory create my-app
```
